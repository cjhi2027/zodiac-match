# 12간지 띠별 궁합 웹앱 개발 문서

## 📋 프로젝트 개요

**프로젝트명**: 12간지 띠별 궁합 웹 애플리케이션  
**목적**: 생년월일 또는 띠 선택을 통한 커플 궁합 확인 서비스  
**타겟**: 한국 및 글로벌 사용자  
**개발 기간**: 2024년 9월 24일 ~ 15일  

---

## 🛠 아키텍처 구조

### **모노레포 구조**
- `client/` - React 프론트엔드 애플리케이션
- `server/` - Express.js 백엔드 (현재 미사용)
- `shared/` - 공통 타입 및 스키마

### **프론트엔드**
- **React + Vite 기반 SPA** (Single Page Application)
- **TypeScript** 완전 지원
- **모바일 우선 반응형 디자인**

### **백엔드**
- **Express.js + Node.js** (현재 미사용)
- **Vercel 배포용** 정적 파일 서빙

---

## 🛠 프론트엔드 기술 스택

### **핵심 프레임워크**
- **React 18.3.1** - UI 라이브러리
- **TypeScript 5.6.3** - 타입 안정성
- **Vite 5.4.19** - 빌드 도구 및 개발 서버
- **ESNext 모듈 시스템** - 최신 JavaScript 모듈

### **UI/스타일링**
- **Tailwind CSS 3.4.17** - 유틸리티 CSS 프레임워크
- **Radix UI** - 접근성 중심 UI 컴포넌트 라이브러리
- **Lucide React** - 아이콘 라이브러리
- **Framer Motion 11.13.1** - 애니메이션 라이브러리
- **class-variance-authority** - 컴포넌트 variant 관리

### **상태 관리 & 폼**
- **React Query (TanStack Query) 5.60.5** - 서버 상태 관리
- **React Hook Form 7.55.0** - 폼 관리
- **Zod 3.24.2** - 스키마 검증

### **라우팅**
- **Wouter 3.3.5** - 경량 라우팅 라이브러리

### **기타 라이브러리**
- **date-fns** - 날짜 처리
- **clsx + tailwind-merge** - 조건부 클래스 관리
- **cmdk** - 명령 팔레트
- **embla-carousel-react** - 캐러셀
- **recharts** - 차트 라이브러리

## 📱 주요 특징

### **사용자 경험**
- **반응형 디자인** (모바일 우선)
- **직관적인 UI/UX** (2단계 간단한 프로세스)

- **생년월일 입력** (자동 띠 계산)
- **궁합 점수 계산** (12x12 매트릭스)
- **점수별 색상 구분** (빨강/노랑/초록/파랑)
- **결과 애니메이션** (카운트업 + 프로그래스바)

### **성능**
- **코드 스플리팅** - 번들 최적화
- **이미지 최적화** - PNG 압축
- **빠른 로딩** - Vite 빌드 최적화

## 🚀 배포 환경

### **프론트엔드 배포**
- **Vercel** - 정적 사이트 호스팅
- **자동 배포** - GitHub 연동
- **CDN** - 글로벌 콘텐츠 전송

### **파일 구조**
- **정적 파일**: `client/public/` 폴더
- **이미지**: `client/public/attached_assets/animal/`
- **빌드 결과**: `client/dist/` 폴더

### **환경 설정**
- **개발**: `npm run dev` (Vite 개발 서버)
- **빌드**: `npm run build` (프로덕션 빌드)
- **미리보기**: `npm run preview` (빌드 결과 확인)
